Gainput

Get your players!

Gainput is the new awesome C++ input library for your game. It's easy to use, open source and cross-platform.

Browse Source API Docs

Nice Code

Gainput's code is clean, easy to understand and lightweight. It's also easy to use and will hopefully make you a happier person.

Cross-Platform

Gainput provides the same interface (more or less) on all supported platforms. Currently, implementations are provided for: Windows, Linux, and Android NDK.

Open Source

Gainput is Open Source and licensed under the MIT license. So feel free to use it, share it, change it however you like.

All the Devices

Gainput aims to support all available input devices. Implementations are provided for keyboard, mouse, pad, and multi-touch surfaces.
